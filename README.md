# software_testing_sgu_2025
A learning process tracking of Software Testing course produced by SaiGon University.


---


## Lab 1:
I learn what is Software Testing. I practice with a C++ unit test example

---

## Lab 2:
I review Analyst & Design a software system

---

## Lab 3:
I learn to design system architecture with C4 Standard Model

---

## Lab 4:
I learn what Test Plan is, and how to write it with a provided example system and it's test plan template.

---

## Lab 5:
I learn what Equivalence Partitioning Testing is.
Practicing with initiate Equivalence Partitioning test cases and analysing the inititation process.

---

## Lab 6:
I practice Black Box Testing method, and initiate test cases with decision table.

---

## Lab 7:
Applying Black Box Testing method to test Process.
